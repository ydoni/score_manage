# 한국대학교 학점관리시스템
한국대학교 학생들의 학점을 등록하는 시스템



**1) 프로젝트 목적**

'React'를 통해 기본적인 CRUD 기능을 구현하도록 한다.
> CRUD 기능 구현 - Create (등록) / Read (읽기) / Update (수정) / Delete (삭제)

**2) 실행화면**
